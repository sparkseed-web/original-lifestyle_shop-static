# オリジナルサイト（架空の雑貨ショップ）

## 概要
HTML/CSS の実践練習として、架空の雑貨ショップのオリジナルサイトを制作しました。

## このサイトを作った目的
- HTML/CSS の実践練習として一から自分の手でサイト制作をする
- HP制作の全体的な流れと、デザイン制作の流れを掴む
- 完成したサイトを　GitHub を利用してサーバーにアップロード（リリース）する

## 使用技術
- HTML
- CSS
- jQuery
-

## こだわったポイント
①トップに動画を大きく配置し、ブランドの空気感やゆっくりと流れる優しい時間が感じられるメインビジュアルにしました。

②安心感や柔らかさが伝わるよう、ゆっくりとした動きや優しい色味を意識してデザインしています。

③ユーザーがクリックしたくなるように、hover時の色の変化や動きをつけたところも工夫した点です。

④休業日を視覚的に伝えるためカレンダーを設置してユーザーが安心して利用できるようにしました。

## 苦労した点　
### - HTMLとCSSの構造を自分で組み立てること  
  管理のしやすいコードにすることを意識していたが、コードが増えるにつれてカオスな状態になってしまいました。  
  
  →→→そのため、どの部分のコードなのかわかるようにコメントアウトを入れたり、主要な色やフォントを :root をつかって変数化するなど、管理しやすいコードに近づける工夫をしました。
  
### - レイアウトが崩れたときの原因の特定  
  デザインカンプと違う場所に要素がズレてしまったときに何が原因で崩れているのかを特定するのに苦労しました。　　
  ⇒⇒⇒検証ツールでズレている要素に意図していない余白がついていないかなどを調べるなどして微調整をしてレイアウトを整えました。

### - レスポンシブ対応の調整  
  スマホサイズにしたときに要素がはみ出してしまったり、余白やフォントサイズが大きすぎたりして微調整が難しかったです。  
  ⇒⇒⇒widthの指定を見直したり、 clamp() を取り入れて、柔軟に変化できるようにして改善を図りました。
